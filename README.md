# Automated-Financial-News-Scraping-and-Structured-Data-Conversion

A Python-based data engineering project that automates the extraction of financial news from web sources and transforms unstructured HTML data into structured datasets for analysis.

---

## 🚀 Overview

Financial news is a valuable source of information for market analysis, sentiment tracking, and decision-making. This project focuses on:

- Scraping financial news articles from websites
- Parsing raw HTML content
- Converting unstructured data into structured formats (JSON/CSV)

The system helps bridge the gap between raw web data and usable datasets for analytics or machine learning applications.

---

## ✨ Features

- 🌐 Automated web scraping of financial news  
- 🧾 Extracts key fields like:
  - Headlines  
  - Dates  
  - Article links  
- 🔄 Converts unstructured HTML → structured data  
- 📁 Saves output in CSV / JSON format  
- ⚙️ Modular and extensible scraping pipeline  

---

## 🏗 Tech Stack

- **Language:** Python  
- **Libraries Used:**
  - BeautifulSoup (HTML parsing)
  - Requests (HTTP requests)
  - Pandas (data handling)
